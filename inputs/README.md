# Inputs

Prospect and client context files for pipeline report builds.

## Structure
- `demo/[firstname]/context.md` - prospect context for DEMO builds (5 leads, no enrichment, conversion CTA)
- `prod/[clientname]/context.md` - client context for LIVE builds (10 leads, enrichment on, no CTA)

Each context.md is generated from the person's LinkedIn profile and contains their ICP, voice, website, and Vibe filters.
